# Recipe Hub

A modern recipe discovery platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Search recipes by name
- Browse recipes by category
- View detailed recipe information
- Responsive design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- TheMealDB API
